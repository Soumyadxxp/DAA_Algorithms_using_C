# DAA Algorithms Using C

A collection of Design and Analysis of Algorithms (DAA) programs implemented in the C programming language. This repository contains fundamental algorithms and data structures commonly taught in undergraduate Computer Science and Engineering courses.

The programs are intended for learning algorithm design, understanding time and space complexity, and preparing for university practical examinations and coding interviews.

---

## About the Repository

This repository includes implementations of various algorithms using C. Each program is written to demonstrate the working of a particular algorithm and can be compiled using any standard C compiler.

## Requirements

- GCC Compiler (or any standard C compiler)
- Code::Blocks, Dev-C++, Visual Studio Code, or any C IDE

---

## Compilation

Compile a program using GCC:

```bash
gcc filename.c -o output
```

Example:

```bash
gcc MERGE_SORT.C -o merge
./merge
```
